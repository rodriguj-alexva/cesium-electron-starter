# cesium-electron-starter
Demo of CesiumJS on Electron, 2020.

## Using

### Clone
        $ git clone https://github.com/rodriguj-alexva/cesium-electron-starter
        
### Install Packages
        $ cd cesium-electron-starter
        $ npm install
        
### Run Electron App
        $ npm start
